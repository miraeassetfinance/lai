Tính Lãi TOPUP

Dựa theo logic file Excel TOPUP • Có ân hạn • Có/không bảo hiểm • So sánh nhiều kỳ hạn

File Excel TOPUP đang dùng logic: Tổng tiền vay = LOAN + INS, EMI ân hạn = (Tổng tiền vay × lãi suất / 365 × số ngày) + 12.000, sau ân hạn dùng PMT theo số tháng còn lại.
